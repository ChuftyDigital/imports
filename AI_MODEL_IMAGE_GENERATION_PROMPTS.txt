## TECHNICAL RECOMMENDATIONS FOR CONSISTENCY

### For All Models:

**CONSISTENCY TIPS:**
1. **Use Same Seed:** When generating multiple images of same character, use identical seed number
2. **Lock Core Features:** Emphasize key identifying features (hair color/style, eye color, skin tone, build)
3. **Reference First Image:** After generating first image, reference it for subsequent images
4. **Batch Generation:** Generate all 3 images in same session when possible
5. **Style Consistency:** Keep same quality/style descriptors across all 3 prompts

**TECHNICAL PARAMETERS:**
- Quality: 4K, photorealistic, professional photography
- Lighting: Natural, flattering, appropriate to setting
- Pose Variety: Frontal, three-quarter, profile OR sitting, standing, close-up
- Expression Range: Keep consistent with character personality
- Background: Simple but contextual to character's niche

**POST-GENERATION TIPS:**
- Select the 3 most consistent images from batch
- Minor editing for final consistency (color matching, lighting balance)
- Create reference sheet with all 3 images side-by-side
- Use these 3 base images for all future content generation of that character
